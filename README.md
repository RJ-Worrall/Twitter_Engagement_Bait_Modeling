# Twitter_Engagement_Bait_Modeling

Introduction:
This project is designed to flag X/Twitter posts meant to provoke engagement, whether it be harmless engagement like asking for likes, or harmful posts meant to incite outrage through fake news, provocatory langauge, and more. The data source is the X/Twitter developer platform, where users can make API calls to receive detailed tweet statistics and metadata. Data was stored in a SQL database, and integrated with Python to clean and organize the data into readable tables. 


Tools Used:  
Python (numpy, pandas, matplotlib,scikit-learn)
SQL
